# Security Policy

## Supported Versions

Security updates are provided for the latest published version of `@noryen/sdk`.
Older versions may not receive fixes.

## Reporting a Vulnerability

Please do not open public GitHub issues for security reports.

Report vulnerabilities privately by emailing `support@noryen.com` with:

- A clear description of the issue
- Steps to reproduce
- Potential impact
- Any suggested remediation

We aim to acknowledge reports within 3 business days and provide regular
updates until resolution.
